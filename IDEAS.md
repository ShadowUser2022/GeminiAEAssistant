# 💡 Gemini AE Assistant — Feature Backlog & Ideas

Этот файл служит копилкой наших технологических идей и улучшений для панели Gemini AE Assistant. Любой новый чат или ассистент может брать задачи отсюда для реализации.

---

## 🚀 Активные идеи для реализации

### 1. Голосовой ввод промптов (Web Speech API Dictation Button)
*   **Проблема:** Встроенная диктовка macOS блокируется After Effects и контейнером CEP из-за перехвата фокуса ввода клавиатуры.
*   **Решение:** Добавить маленькую неоновую кнопку-микрофон прямо в поле ввода чата.
*   **Реализация:**
    *   При клике запускать встроенное в Chromium браузерное распознавание `webkitSpeechRecognition`.
    *   Отображать пульсирующий индикатор записи в поле ввода.
    *   По окончании речи автоматически вставлять распознанный текст в `promptInput`.
    *   *Плюсы:* Работает на 100% кроссплатформенно, не зависит от фокуса macOS/Windows и не требует внешних библиотек.

### 2. Визуальное превью прикрепленного контекста
*   **Идея:** При наведении на чипс-ссылку выделенного слоя (в нашем новом `context-chips-container`) показывать маленькую всплывающую карточку с дополнительной информацией о слое:
    *   Тип слоя (Text, Shape, Solid, Footage).
    *   Время старта/конца на таймлайне.
    *   Активные эффекты, примененные к слою.
*   **Реализация:** Расширить ExtendScript функцию `getSelectedContext()`, чтобы она собирала эти метаданные, и выводить их в HTML-атрибуты чипса.

### 3. Интеллектуальный поиск по истории (Smart Log Viewer)
*   **Идея:** Добавить вкладку или выдвижную панель со списком прошлых сгенерированных скриптов.
*   **Реализация:** Читать файлы логов из папки `/logs` и выводить красивый интерактивный список с кнопкой «Применить повторно».
